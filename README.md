# 🚀 First_my_app

مرحباً بك في مشروعي الأول المعتمد على تقنيات البرمجة الحديثة!  
Welcome to my first project using modern development technologies!

---

## 📝 عن المشروع (About)
هذا التطبيق هو أول تجربة لي في بناء تطبيقات الويب باستخدام **Kotlin Multiplatform (KMP)** و **Compose Multiplatform**. تم تحويل الكود المصدري إلى **WebAssembly (Wasm)** لضمان أداء عالٍ وتجربة مستخدم سلسة مباشرة في المتصفح.

This is my first experiment in building web applications using **Kotlin Multiplatform (KMP)** and **Compose Multiplatform**. The source code is compiled into **WebAssembly (Wasm)** to ensure high performance and a smooth user experience directly in the browser.

## 🛠️ التقنيات المستخدمة (Tech Stack)
* **Language:** Kotlin 
* **UI Framework:** Compose Multiplatform
* **Target:** Web (Wasm / JS)
* **Rendering Engine:** Skia (Skiko)

## 🏗️ كيف يعمل؟ (How it works)
يعتمد التطبيق على محرك **Skia** للرسم وتوليد الواجهات، ويستخدم **Wasm** لتشغيل منطق البرمجة بسرعة تقارب تطبيقات سطح المكتب.

The app uses the **Skia** engine for UI rendering and leverages **Wasm** to run logic at near-native speeds.
